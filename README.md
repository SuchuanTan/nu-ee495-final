# Double Pendulum — ML Method Comparison

This project compares four machine learning approaches for learning and predicting the dynamics of the double pendulum, a canonical chaotic system with a conserved Hamiltonian.

## What This Project Does

Ground-truth trajectories are generated by numerically integrating the exact equations of motion. Four models are then trained on this data and evaluated on three criteria: one-step prediction accuracy, long-term rollout error, and energy conservation.

| Model | Key Idea |
|---|---|
| **MLP** | Baseline: directly maps state at time t to state at time t+1 with no physical assumptions |
| **ResNet** | Residual connections bias the network toward learning state increments |
| **Neural ODE** | Learns a continuous vector field, integrates forward with RK4 |
| **HNN** | Learns a scalar Hamiltonian, derives dynamics via autograd and Hamilton's equations |

## Results

All figures are in the `images/` folder:

- `sample_trajectory.png` — example ground-truth trajectory and energy profile
- `training_loss.png` — training loss curves for all four models
- `short_term_bar.png` — one-step MSE comparison
- `rollout_mse.png` — long-term rollout error over time
- `energy_error.png` — energy conservation error over time
- `trajectory_comparison.png` — predicted vs ground-truth trajectories

Key finding: Neural ODE achieves the best short-term accuracy. HNN underperforms on energy conservation under limited training, despite its structural guarantee — this is consistent with known data requirements of Hamiltonian networks.
